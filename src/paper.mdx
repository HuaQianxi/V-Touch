## Abstract

We introduce **VTOUCH**, a large-scale multimodal dataset for bimanual robot manipulation featuring synchronized proprioception, multi-view RGB-D observations, and high-resolution fingertip tactile signals from GelSight sensors. The dataset covers 380+ bimanual tasks with 100+ atomic action compositions, collected across multiple robot embodiments.

<Figure>
  <Picture slot="figure" src={datasetView} alt="Dataset Overview" />
  <Fragment slot="caption">Dataset Overview. Multi-modal bimanual manipulation dataset with synchronized proprioception, multi-view RGB-D, and fingertip tactile signals.</Fragment>
</Figure>

Our dataset addresses the critical gap in existing benchmarks by jointly capturing visual, tactile, and proprioceptive signals from real robot hardware, enabling research on contact-rich bimanual manipulation.

## Dataset Comparison

We compare VTOUCH against existing datasets covering human motion capture, synthetic robotics, and real robot manipulation. As shown below, none of the existing datasets provide all three key features: dual-arm support, real tactile sensing, and physical validation.

## Data Acquisition System

To enable cross-embodiment data collection, we built a unified hardware abstraction interface that aligns different robot platforms at the state, action, and sensing levels.

<Figure>
  <Picture slot="figure" src={crossEmbodiment} alt="Cross-Embodiment Data Collection" />
  <Fragment slot="caption">Cross-embodiment data collection system supporting fixed dual-arm, wheeled-arm, and UMI-style mobile manipulators.</Fragment>

The system integrates head-mounted and wrist-mounted RGB-D cameras along with GelSight fingertip tactile sensors, all synchronized via a unified hardware triggering module achieving precise temporal alignment across modalities.

## Task Framework

We propose a skill-axis classification framework that characterizes tasks along six orthogonal dimensions: bimanual coordination structure, atomic action types, contact modes, object geometry, perception requirements, and task hierarchy.

## Cross-Modal Retrieval Results

### Method Overview

We evaluate cross-modal retrieval performance using three modalities: Visual (V), Tactile (T), and Pose (P). Our contrastive learning framework projects all modalities onto a shared embedding space and optimizes alignment via InfoNCE loss.

Bimodal retrieval tests single modality queries across 6 directions: V↔T, V↔P, T↔P. Our method significantly outperforms CCA and PLSCA baselines.

## Real-Robot Validation

### 4-Layer Progressive Validation Framework

We validate policies using a four-layer progressive validation strategy before real-world deployment:

1. **Layer 1**: Action Reconstruction — verifies policy can reproduce expert actions
2. **Layer 2**: Closed-Loop Validation — checks physical validity and smoothness
3. **Layer 3**: Short-Horizon Rollout — tests temporal consistency
4. **Layer 4**: Consistency Evaluation — measures output variance

### Validation Results

Results on OpenLoong platform using ACT policy with 3 observation steps:

| Layer | Metric | Value |
|-------|--------|-------|
| Layer 1: Reconstruction | MAE | 0.677 |
| Layer 1: Reconstruction | Expert Similarity | 0.445 |
| Layer 2: Smoothness | Action Diff Mean | 0.007 |
| Layer 3: Rollout | Final Error | 0.690 |
| Layer 3: Rollout | Error Growth | -0.014 |
| Layer 4: Consistency | Mean Variance | 0.000283 |
| **Overall** | **Score** | **0.778 (Fair)** |

*Validation results on OpenLoong platform with ACT policy (n_obs_steps=3).*

The negative error growth (-0.014) indicates stable short-horizon behavior.

## Conclusion

VTOUCH provides the first large-scale, multimodal bimanual manipulation dataset with synchronized RGB-D, tactile, and proprioceptive data collected from real robot hardware. Key contributions:

- **380+ bimanual tasks** with 100+ atomic action compositions
- **Cross-embodiment data collection** across Qinglong, wheeled-arm, and UMI platforms
- **Skill-axis classification framework** for systematic task organization
- **Cross-modal retrieval** achieving 2-3x improvement over baselines
- **4-layer validation framework** for systematic policy evaluation

The dataset and benchmarks enable research toward robust, generalizable bimanual manipulation in real-world environments.
