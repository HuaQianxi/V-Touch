## Abstract

Embodied intelligence has advanced rapidly in recent years; however, bimanual manipulation—especially in contact-rich tasks—remains challenging. This is largely due to the lack of datasets with rich physical interaction signals, systematic task organization, and sufficient scale. To address these limitations, we introduce the **VTOUCH** dataset. It leverages vision-based tactile sensing to provide high-fidelity physical interaction signals, adopts a matrix-style task design to enable systematic learning, and employs automated data collection pipelines covering real-world, demand-driven scenarios to ensure scalability. To further validate the effectiveness of the dataset, we conduct extensive quantitative experiments on cross-modal retrieval as well as real-robot evaluation. Finally, we demonstrate real-world performance through generalizable inference across multiple robots, policies, and tasks.

## Data Acquisition System

To enable cross-embodiment data collection, we built a unified hardware abstraction interface that aligns different robot platforms at the state, action, and sensing levels.

<Figure>
  <Picture slot="figure" src={crossEmbodiment} alt="Cross-Embodiment Data Collection" />
  <Fragment slot="caption">Cross-embodiment data collection system supporting fixed dual-arm platforms, wheeled-arm systems, and UMI-style mobile manipulators. All platforms are connected via a unified hardware abstraction interface that semantically aligns different hardware configurations at the state, action, and sensing levels.</Fragment>

The system integrates head-mounted and wrist-mounted RGB-D cameras along with GelSight fingertip tactile sensors, all synchronized via a unified hardware triggering module achieving precise temporal alignment across modalities.

## Task Framework

We propose a skill-axis classification framework that characterizes tasks along six orthogonal dimensions: bimanual coordination structure, atomic action types, contact modes, object geometry, perception requirements, and task hierarchy.

<Figure>
  <Picture slot="figure" src={taskClass} alt="Task Classification Framework" />

This structured representation enables generation of diverse task instances through flexible axis combinations, supporting both fine-grained skill transfer and generalization analysis.

## Cross-Modal Retrieval Results

### Method Overview

We evaluate cross-modal retrieval performance using three modalities: Visual (V), Tactile (T), and Pose (P). Our contrastive learning framework projects all modalities onto a shared embedding space and optimizes alignment via InfoNCE loss.

## Real-Robot Validation

### 4-Layer Progressive Validation Framework

We validate policies using a four-layer progressive validation strategy before real-world deployment:

1. **Layer 1**: Action Reconstruction — verifies policy can reproduce expert actions
2. **Layer 2**: Closed-Loop Validation — checks physical validity and smoothness
3. **Layer 3**: Short-Horizon Rollout — tests temporal consistency
4. **Layer 4**: Consistency Evaluation — measures output variance

### Validation Results

We validate three policy implementations on the VTouch dataset: ACT with single-frame observation, ACT with temporal context, and a diffusion-based policy. The diffusion policy achieves the best overall performance with the lowest MAE (0.022) and highest Expert Similarity (0.848), while the temporal ACT model shows negative error growth (-0.010), indicating stable short-horizon behavior.

| Metric | ACT (base) | ACT (temporal) | Diffusion Policy |
|--------|:----------:|:--------------:|:----------------:|
| n_obs_steps | 1 | 3 | 1 |
| MAE | 0.516 | 0.709 | 0.022 |
| Expert Similarity | 0.577 | 0.435 | 0.848 |
| Action Diff Mean | 0.165 | 0.008 | 0.044 |
| Final Error (Layer 3) | 0.500 | 0.691 | 0.431 |
| Error Growth (Layer 3) | 0.0003 | -0.010 | 0.0002 |
| Mean Variance (Layer 4) | 0.0001 | 0.0002 | 0.0001 |
| **Overall Score** | **0.674** | **0.774** | **0.836** |

*Validation results on OpenLoong platform. Negative error growth for ACT (temporal) indicates stable short-horizon behavior.*

## Conclusion

VTOUCH provides the first large-scale, multimodal bimanual manipulation dataset with synchronized RGB-D, tactile, and proprioceptive data collected from real robot hardware. Key contributions:

- **380+ bimanual tasks** with 100+ atomic action compositions
- **Cross-embodiment data collection** across Qingloong, Wheelloong, and UMI platforms
- **Skill-axis classification framework** for systematic task organization
- **Cross-modal retrieval** achieving 2-3x improvement over baselines
- **4-layer validation framework** for systematic policy evaluation across multiple policies

The dataset and benchmarks enable research toward robust, generalizable bimanual manipulation in real-world environments.
